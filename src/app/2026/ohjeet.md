
## Windows

**Vaatimukset**

- Lataa ja asenna Modrinth App: [https://modrinth.com/app](https://modrinth.com/app)

- Lataa tiedosto yllä olevasta napista

**Ohjeet**

1. Kirjaudu microsoft tilillesi Modrinth sovelluksessa
2. Luo uusi instanssi vasemmalla näkyvästä plus napista
	- Valitse from file ja valitse lataamasi tyoleiri_2026 -tiedosto
3. Vasemmalla valikossa olevassa **Library** näkymässä näet asennetun instanssin.
4. Avaa peli klikkaamalla kuvaketta ja lisää työleiri.fi palvelin normaalisti

## Mac

_Tällä ohjeella voit asentaa myös Windowsille jos valitset ladatessa windows version_

**Vaatimukset**

- Lataa ja asenna Prism Launcher (universal): [https://prismlauncher.org/download/mac/](https://prismlauncher.org/download/mac/)

- Lataa tiedosto yllä olevasta napista

**Ohjeet**

1. Kirjaudu microsoft tilille 
2. Lisää uusi instanssi klikkaamalla "Lisää pelipaketti" 
3. Valitse "Tuo"
    - Lisää lataamasi tiedosto klikkaamalla selaa ja valitsemalla ladattu tiedosto
4. Käynnistä peli ja lisää työleiri.fi palvelin normaalisti

### Lisätietoa

Aiemmin lisätyt servut, maailmat tai asetukset ei näy tämän kautta pelatessa mutta ne on edelleen säilössä muualla. Pääset niihin käsiksi avaamalla pelin normaali Minecraft Launcherin kautta
